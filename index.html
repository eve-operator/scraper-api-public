🧭 EveScraper API v1.1 — LIVE

Scrape any website with a single HTTP request. Extract emails, page text, links, headings, and more. Ready-to-use API for developers, marketers, researchers.

🚀 Quick Start

curl "http://2a02:6ea0:c88e:5:0:345e:a4f:1/scrape?url=https://example.com&selector=emails&key=YOUR_KEY"

Replace selector with: text, title, emails, links, headings

📊 API Endpoints

  • GET /scrape — Single URL extraction
    /scrape?url=URL&selector=emails|text|title|...&key=KEY
  • GET /bulk-scrape — CSV batch processing
    /bulk-scrape?csv=CSV_URL&key=KEY
  • GET /health — Service status
  • GET /pricing — Pricing JSON
